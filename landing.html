03 // WHY IT WORKS — BUILT TO COMPOUND, NOT GAMBLE
🌧
3-Model Consensus
GFS (31 members) and ECMWF (51 members) are combined into a single weighted probability. Four confidence tiers — LOCK, STRONG, SAFE, NEAR-SAFE — determine how aggressively to bet. No single model, no guessing.
📈
Kelly Sizing
Every trade is sized by the Kelly criterion to maximize long-term growth. Quarter-Kelly keeps variance low. LOCK signals bet 1.5×, STRONG 1.2×. Max position capped at 5% of bankroll — one bad trade can’t blow you up.
⚡
Circuit Breaker
3 consecutive losses and trading halts automatically. No emotional revenge trading, no death spiral. Resets on the next win. Your bankroll is protected even when the models miss.
📄
Paper → Live Toggle
Start in paper mode with zero risk — identical logic, no real money. When you’re confident, flip to live and ElementClaw places real CLOB orders on Polymarket. One environment variable, same codebase.
⚙
OpenClaw Deployment
Install with openclaw skill add elementclaw and it runs on your VPS via cron. No Docker, no Kubernetes. Scans every 30 minutes, settles daily, reports via web dashboard.
📊
Live Web Dashboard
Real-time P&L, open positions, signal history, and settlement status in a single dashboard. See exactly what the bot is doing, what it’s holding, and how much you’ve made — from any device.
04 // THE EDGE — WEATHER MODELS KNOW MORE THAN THE MARKET DOES
Step 01
The Market
Polymarket lists daily temperature brackets for 6 US cities — “Will NYC’s high be 38–39°F?” Each bracket trades like a stock with a price between 0¢ and 100¢. Correct predictions pay $1 USDC. That price is set by the crowd — and the crowd is often wrong.
Step 02
The Mispricing
ElementClaw fetches 122 ensemble forecast members from GFS and ECMWF — the same models used by NOAA. It calculates the true probability of each bracket and compares it to the market price. When the gap exceeds 8%, that’s a trade.
Step 03
The Execution
Each signal is sized using the Kelly criterion to maximize long-term growth without risking ruin. Paper mode logs the trade instantly. Live mode places a real limit order on Polymarket’s CLOB. Every 30 minutes, the bot scans again.
Step 04
The Settlement
When the day ends, actual observed temperatures are fetched from Iowa State Mesonet. Winning positions pay $1 USDC per share. Losses cost your entry price. P&L is tracked automatically — you just check the dashboard.

07 // DOCUMENTATION
Quick Start
Install & Run
1. Install OpenClaw on your VPS
2. Run openclaw skill add elementclaw
3. Set your POLYGON_PRIVATE_KEY and BANKROLL_USDC in ~/.openclaw/openclaw.json
4. Start with MODE=paper to test
5. Switch to MODE=live when ready
Commands
CLI Reference
bun run paper — Continuous loop for local dev

bun run scan — Fetch weather, scan markets, trade

bun run settle — Check settlements, update P&L

bun run status — JSON report of positions + stats

bun test — Run the test suite
Configuration
Environment Variables
MODE — "paper" or "live"

BANKROLL_USDC — Starting capital (default $50)

MIN_EDGE_PCT — Minimum edge threshold (default 8%)

KELLY_FRACTION — Sizing aggressiveness (default 0.25)

MAX_OPEN_POSITIONS — Max concurrent trades (default 10)
Architecture
How It’s Built
Stack: TypeScript + Bun + SQLite
Runtime: OpenClaw cron on VPS
Data: Open-Meteo (GFS/ECMWF ensembles), NWS point forecasts, Iowa State Mesonet (settlement)
Trading: Polymarket CLOB via EIP-712 + HMAC
Dashboard: Bun.serve() on port 3456
Cron Schedule
Production Timing
Scan: Every 30 minutes
Fetches forecasts, discovers markets, finds edge, executes trades

Settle: Hourly
Checks actual temps from Iowa State, resolves positions, updates P&L

Status: Daily at 8 AM
JSON report of all positions and performance stats
Risk Controls
Safety Mechanisms
Circuit Breaker: 3 consecutive losses halts trading

Quarter-Kelly: Conservative sizing limits per-trade risk

Max Position: 5% of bankroll cap per trade

Max Open: 10 concurrent positions limit

Edge Threshold: Only trades with 8%+ edge execute
08 // QUESTIONS
How much money can I make?
It depends on your bankroll and market conditions. Backtest data shows a 61.7% win rate with an average edge of 20.7% per trade. With a $50 bankroll and quarter-Kelly sizing, individual trades are $1–$3 USDC. Compounding winners grows the bankroll over time. This is not a get-rich-quick scheme — it’s a systematic edge that compounds.
What’s the minimum to start?
$50 USDC on Polygon is a practical minimum. Individual trades range from $1–$3 with default settings. You’ll also need a small amount of MATIC for gas fees (a few cents per transaction). Paper mode is completely free — no wallet or funds required.
Is this actually profitable or just backtested?
The backtests use real historical market prices and real ensemble forecast data — no lookahead bias. Paper mode lets you verify the edge in real-time before risking a dollar. The signal table on this page shows real output from a live scan. That said, past performance doesn’t guarantee future results and market conditions can change.
What if it starts losing money?
The circuit breaker automatically halts all trading after 3 consecutive losses. No emotional revenge trading, no death spiral. It resets on the next win. Quarter-Kelly sizing also ensures no single trade risks more than a small fraction of your bankroll. You can also pause the bot at any time by stopping the OpenClaw cron.
Do I need to know anything about weather or trading?
No. ElementClaw handles everything — fetching forecasts, discovering markets, calculating edge, sizing positions, and settling outcomes. You install it, set your bankroll, and let it run. The dashboard shows you exactly what’s happening in plain English. If you want to understand the math, the code is open source.
What is Polymarket?
Polymarket is a prediction market platform where you buy and sell shares on real-world outcomes. Weather markets let you bet on questions like “Will NYC’s high be between 38–39°F tomorrow?” Shares trade between 0¢ and 100¢. If you’re right, your shares pay out $1 USDC each. If wrong, they expire worthless. It’s built on Polygon and settles in USDC.
